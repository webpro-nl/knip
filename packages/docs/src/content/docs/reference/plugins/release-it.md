## Enabled

This plugin is enabled when there is match in `dependencies` or
`devDependencies`:

- `release-it`

## Default configuration

```json
{
  "release-it": {
    "config": [
      ".release-it.json",
      ".release-it.{js,cjs}",
      ".release-it.{yml,yaml}",
      "package.json"
    ]
  }
}
```
